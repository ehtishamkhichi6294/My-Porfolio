# Ehtisham Khichi - Portfolio Website

A modern, responsive, and high-performance portfolio website built with React, Vite, Tailwind CSS, and Framer Motion. The website features a sophisticated dark theme with red accents, advanced glassmorphism effects, and smooth page transitions.

## 🌐 Live Preview
[View Live Site](https://ehtishamkhichi.netlify.app)

## ✨ Features

- **Modern UI/UX:** Sleek dark theme with red accents, glassmorphism, and glowing hover effects.
- **Smooth Animations:** Page transitions and scroll-reveal animations using Framer Motion.
- **Fully Responsive:** Mobile-first design ensuring a seamless experience across all devices.
- **Dynamic Resume Generation:** On-the-fly PDF resume generation using `jspdf`.
- **Spam-Protected Contact Form:** Integrated with Formspree, featuring a Math Captcha and Honeypot field for bot protection.
- **SEO Optimized:** Dynamic meta tags, Open Graph, and Twitter card data using `react-helmet-async`.
- **Performance Focused:** Lazy loading for images and optimized asset delivery.

## 🛠️ Tech Stack

- **Frontend Framework:** React 18
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion (`motion/react`)
- **Routing:** React Router DOM
- **Icons:** Lucide React
- **PDF Generation:** jsPDF
- **Form Handling:** Formspree

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/ehtishamkhichi6294/portfolio.git
   ```

2. Navigate to the project directory:
   ```bash
   cd portfolio
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Build for production:
   ```bash
   npm run build
   ```
